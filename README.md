# Lantero — Family Learning Planner

A private, **on-device** planner for parent-led family learning. "Warm light on old paper, pointed at any when, any where."

- **Single-file web app** (`index.html`) — vanilla JS, no build step, no backend.
- **Local-first:** all data lives in the browser's `localStorage`. Nothing is sent anywhere. Export a backup anytime from **Settings → Data**.
- **Parent-facing only** by design.

## Run / host
Open `index.html` in any modern browser, or serve the folder as a static site (this repo is published via GitHub Pages). HTTPS is recommended so it can be installed to a phone's home screen (manifest + icon included; no offline service worker yet).

## Feedback
Use the **Send feedback** button in the header (opens an email), or export your backup JSON and attach it.
